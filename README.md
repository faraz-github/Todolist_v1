# Todolist_v1

A simple Todolist like app made using Nodejs environment with the MongoDB database.

## Available Commands

- npm install
- node app.js

## Installation

Todolist_v1 requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
cd Todolist_v1
npm i
node app.js
```
